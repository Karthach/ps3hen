# Contribuir al proyecto (en español)

Gracias por contribuir. Sigue estas indicaciones básicas:

1. Haz un fork y trabaja en una rama (feature/mi-cambio).
2. Mantén commits pequeños y con mensajes claros en español.
3. Añade documentación para cambios visibles.
4. Abre un Pull Request con descripción clara y referencia a issues.

Para publicar binarios pesados se recomienda usar GitHub Releases y no almacenar archivos binarios grandes en el repositorio.
